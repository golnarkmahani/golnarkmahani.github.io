On the Concept of Topological Data Analysis

By Golnar Mahani February 1, 2019

The main goal of Topological Data Analysis (TDA) is to use ideas and results from geometry and topology to develop tools for studying qualitative features of complex data. The topological and geometric information provides new features of the data; and these features can be used for a better understanding of the data by visualisation or can be used in a second stage of analysis using machine learning techniques. In this talk, I introduced the fundamental concepts of TDA and an open discussion on the possible applications of it to NeuroImaging research.
